HTML5 带来了什么？

1. 更加语义化的标签。如 article, header, footer, nav 等。
2. 更丰富的媒体支持：video 和 audio。
3. 更好的绘图：canvas 和 svg。
4. 更好的存储和离线缓存。（localStorage, sessionStorage）
5. 更强的计算能力。（worker）
6. 服务器通信：web sockets
