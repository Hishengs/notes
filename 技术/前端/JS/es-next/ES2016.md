## Exponentiation (**)

https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Exponentiation

```js
console.log(3 ** 4);
// expected output: 81

console.log(10 ** -2);
// expected output: 0.01

console.log(2 ** 3 ** 2);
// expected output: 512

console.log((2 ** 3) ** 2);
// expected output: 64
```
