### 时间复杂度

我们常用大 O 表示法来表示一个算法的时间复杂度，即随着输入规模的增大，与算法所需要时间之间的关系，一般有以下一种：

- `Θ(1)` 常数级别
- `Θ(log(n))` 对数级别
- `Θ(n)` 线性级别
- `Θ(n*log(n))` 线性对数级别
- `Θ(n^2)` 平方级别
- `Θ(2^n)` 指数级别
