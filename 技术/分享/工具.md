分享一些好用、实用的工具



**[Typora](https://typora.io)**

一款简洁而强大的 markdown 编辑器。



**Rapidee**

一款在 windows 环境下用于设置环境变量的工具。



**[postman](https://www.getpostman.com/)**

一个用发起 HTTP 请求，API 测试的工具，前端开发必备。



**[火绒](http://huorong.cn/)**

一个简单，不流氓的安全软件。



**Xshell**

用于连接远程 Linux 的强大、美观的命令行工具。



**Snipaste**

windows 下的截图工具。
